1.安装java sdk 

2.安装 nodejs 

  版本6.0~7.0最佳
  
  
3.安装 python 

  版本2.7最佳 
  
  下载地址[python](http://www.python.org/download)
  
4.下载并安装 Android Studio(AS) 

  版本2.2.0.12
  
5.命令行安装如下库 

  npm i -g react-native-cli
